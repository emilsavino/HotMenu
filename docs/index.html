HotMenu

A tiny macOS menu bar app for CPU temperature, fan speed, CPU load and memory usage.

Download

Download for macOS (.dmg) or .zip

Latest version: latest · macOS 26 or newer · Apple Silicon

How to install

HotMenu isn't signed by Apple (which costs $99 a year), so macOS will block it the first time you try to open it. You only have to do this once.

  1. Download the .dmg above and open it.
  2. Drag HotMenu into the Applications folder.
  3. Open the Terminal app (press ⌘ Space, type "Terminal", press Return).
  4. Paste this command and press Return: 
    xattr -r -d com.apple.quarantine /Applications/HotMenu.app
  5. Open HotMenu from your Applications folder. A small flame icon will appear in your menu bar.

If you skip step 4, macOS will say "HotMenu can't be opened because Apple cannot check it for malicious software." The command above tells macOS that you trust this app.

What it shows

  • CPU temperature in the menu bar
  • Fan speed (on Macs that have fans)
  • CPU usage and memory usage when you click the menu bar icon
